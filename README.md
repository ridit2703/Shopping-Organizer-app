# 🛒 Shopping Organizer

A simple and interactive **Shopping Organizer Web App** built using **HTML, CSS, and JavaScript DOM manipulation**.
This app helps users manage shopping items efficiently by organizing them into different lists.

---


## 📌 Features

* ➕ Add new shopping items
* 📝 Edit items (double-click to edit)
* ❌ Delete items instantly
* 🖱️ Drag & drop items between lists
* 📂 Organize items into:

  * To Buy List
  * Not Found List
  * Bought List
* 🔄 Reset all lists with confirmation
* ✅ Complete button with feedback message

---
